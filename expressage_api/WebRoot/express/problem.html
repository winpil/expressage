快递一哥支持几种支付方式？
您好！快递一哥支付两种支付方式，一种是现金支付，这种支付方式只需把费用交给收单的快递员即可。另一种支付方式 是微信支付。用户在快递收单时填完单，快递元会给用户一个二维码扫描，用户只需扫一扫就可以支付费用。
